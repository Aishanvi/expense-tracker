# Expense Tracker (Python)

A simple command-line application to track daily expenses.

## Features
- Add expenses
- View all expenses
- Calculate total spending
- Persistent storage using file handling

## Technologies Used
- Python
- File Handling

## How to Run
1. Open terminal
2. Navigate to project folder
3. Run:
   python expense_tracker.py

## Project Purpose
This project demonstrates basic Python concepts like functions, loops, and file handling.
